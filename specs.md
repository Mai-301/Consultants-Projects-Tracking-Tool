Consultants Projects Tracking Tool

Users:

1-Admins
2-Consultants

User stories:

1- As an authorized admin, I want to log into the application.
2- As an admin, I can view the current working project, its status, its employees’ statuses, and the projects' progress reports.
3- As an admin, I can modify on some project's data like: deadline, add/delete tasks, and immediately warns me if there is a new added task out of the new timeframe
4- As an admin, I want to fill in a new project form, which includes: project name, department name (LOV), assigned employees (LOV), project deadline, project start date…etc.
5- As an authorized consultant, I want to log into the application.
6- As a consultant, I want to receive a notification for each assigned task.
7- As a consultant, I want to approve, decline, delay…etc the assigned tasks.
8- As a consultant, I want to change the tasks’ statuses once I start/finish/working on a task.
9- As an admin, I want to view a chart for each assigned consultant once he/she approved his/hers tasks.
10- As an admin, I want to track daily each prject progress by displaying a chart for the project as long as the consultants work on the project’s tasks.
11- As an admin, I want to track all projects monthly by displaying a monthly progress report “Chart” for each project.
12- As an admin, I want to get results from these charts, so I can analyze which attribute is the most effective one.
